# Импорт товаров (CSV, UTF-8)

Колонки (первая строка — заголовок, регистр не важен):

| Колонка | Обязательно | Описание |
|---------|-------------|----------|
| `name` | да | Наименование |
| `price` | да | Цена продажи (число, запятая или точка) |
| `quantity` | нет | Остаток, по умолчанию 0 |
| `description` | нет | |
| `category` | нет | |
| `manufacturer` | нет | |
| `supplier` | нет | |
| `unit` | нет | Единица измерения |
| `discount_percent` | нет | 0–100 |
| `base_price` | нет | Базовая цена (выше цены продажи — зачёркивается в списке) |
| `image_url` | нет | `http(s)://…` или пусто |

Пример: `demo_products.csv`.
